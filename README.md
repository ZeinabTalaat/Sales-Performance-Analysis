## 📌 Project Overview

This project is an end-to-end **Sales Performance Analysis** solution designed to transform raw sales data into meaningful business insights.

The project follows the complete data analytics workflow:

> **Raw Data → Data Cleaning → Data Modeling → DAX Measures → Interactive Dashboards → Business Insights**

The analysis was implemented using **Power Query, Power BI, DAX, Microsoft Excel, and Tableau**.

---

## 🎯 Project Objectives

- Analyze overall sales performance.
- Identify top-performing products and categories.
- Analyze customer purchasing behavior.
- Compare supplier and manufacturing country performance.
- Analyze sales trends over time.
- Evaluate payment methods and bank contributions.
- Compare store and regional performance.
- Create interactive dashboards for business decision-making.

🧹 Data Cleaning – Power Query

The raw dataset was cleaned and transformed using Power Query.

Main Cleaning Steps
Removed duplicate records.
Handled missing and null values.
Corrected data types.
Standardized text values.
Formatted date fields.
Created time-related columns.
Removed data errors.
Prepared the dataset for analysis.
🧩 Data Modeling

A structured data model was created to organize the data efficiently.

The model includes:

Fact Table
Customer Dimension
Item Dimension
Store Dimension
Date Table
Time Dimension
Transaction Dimension

A Star Schema approach was used to improve data organization and analytical performance.
