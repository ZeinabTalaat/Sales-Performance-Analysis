## 📌 Project Overview

This project is an end-to-end Sales Performance Analysis solution that transforms raw sales data into meaningful business insights.

The project covers the complete Data Analytics workflow, from data cleaning and transformation to data modeling, DAX calculations, interactive dashboards, and business insights.

---

## 🎯 Project Objectives

- Analyze overall sales performance.
- Identify top-performing products and categories.
- Analyze customer purchasing behavior.
- Evaluate supplier and country performance.
- Analyze sales trends over time.
- Understand payment methods and bank contributions.
- Compare regional and store performance.
- Build interactive dashboards for data-driven decision-making.

## 🧹 Data Cleaning – Power Query

The raw dataset was cleaned and transformed using Power Query.

### Main Cleaning Steps

- Removed duplicate records.
- Handled missing and null values.
- Corrected data types.
- Standardized text values.
- Formatted date fields.
- Created time-related columns.
- Removed errors.
- Prepared the dataset for analysis.

---

## 🧩 Data Modeling

A structured data model was created using a Star Schema approach.

### Main Tables

- Fact Table
- Customer Dimension
- Item Dimension
- Store Dimension
- Date Table
- Time Dimension
- Transaction Dimension

The data model was designed to improve data organization and support efficient analysis.

---

## ⚡ DAX Measures

DAX was used to create business calculations and analytical measures.

### Main Categories

- Main KPI Measures
- Year-over-Year Measures
- Previous Year Measures
- Ranking Measures
- Top N Analysis
- Running Total Measures
- Color Logic Measures

These measures were used throughout the Power BI dashboards.

# 📊 Power BI Dashboards

The project includes 9 interactive Power BI dashboard pages.

### 1️⃣ Executive Overview

Provides a high-level summary of the most important business KPIs and overall performance.

### 2️⃣ Sales Analysis

Analyzes revenue performance, sales trends, and the strongest contributors to total sales.

### 3️⃣ Time Performance

Analyzes sales performance across months and different time periods to identify trends and seasonal patterns.

### 4️⃣ Product Analysis

Analyzes product performance, top-selling products, sales contribution, quantity sold, and pricing.

### 5️⃣ Customer Analysis

Analyzes customer activity, purchase behavior, customer value, and customer-level patterns.

### 6️⃣ Store Performance

Compares stores or branches and identifies high-performing and low-performing locations.

### 7️⃣ Payment Analysis

Analyzes payment methods, transactions, and their contribution to total sales.

### 8️⃣ Supplier Analysis

Evaluates supplier performance and their contribution to overall sales.

### 9️⃣ Manufacturing Country Analysis

Analyzes the contribution of different manufacturing countries to product sales and supply performance.
